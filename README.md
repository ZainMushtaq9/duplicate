# SEO/GSO Search Index Keyword Site

Production-ready Next.js implementation for a superior MEPCO/PITC utility website based on the strategy in `MEPCO_SEO_GSO_Backlink_Report.docx`.

## Includes

- Next.js App Router
- TypeScript
- Tailwind CSS
- SEO metadata
- Open Graph and Twitter cards
- Organization, FAQ, WebApplication, and Breadcrumb schema
- `sitemap.xml`
- `robots.txt`
- `llms.txt`
- MEPCO bill checker and printable bill result route
- Company duplicate bill pages
- PITC services silo pages

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
